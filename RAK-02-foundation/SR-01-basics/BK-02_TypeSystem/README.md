# BK-02: Type System (The Physics of Data)

Buku ini membedah sifat fisik data di memori, dari presisi numerik hingga representasi teks Unicode.

## 🏁 Daftar Bab

### 1. [CH-01: Numeric Types](./CH-01_NumericTypes/)
Integers, Floats, dan bagaimana arsitektur CPU memengaruhi ukuran tipe data.

### 2. [CH-02: String, Rune, and Byte](./CH-02_StringRuneByte/)
Deep dive ke dunia UTF-8, *immutability*, dan perbedaan karakter vs byte.

### 3. [CH-03: Type Defs and Aliases](./CH-03_TypeDefsAndAliases/)
Perbedaan antara membuat tipe baru (*Definition*) dan label baru (*Alias*).

### 4. [CH-04: Conversions](./CH-04_Conversions/)
Mekanisme konversi eksplisit, *truncation*, dan bahaya *overflow*.

### 5. [CH-05: Pointers Intro](./CH-05_PointersIntro/)
Pengenalan alamat memori, operator `&` dan `*`, serta keamanan `nil`.
