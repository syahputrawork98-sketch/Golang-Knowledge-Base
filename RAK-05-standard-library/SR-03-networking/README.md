# SR-03: Networking

Sub-rak ini membahas kemampuan standard library Go untuk komunikasi jaringan, terutama pada area web dan HTTP.

## Struktur

### [BK-01-web-api](./BK-01-web-api/)
`net/http`, server, client, request-response cycle, dan dasar web API di Go.

## Boundary

- fokus pada networking yang dibangun langsung dari package bawaan;
- relevan untuk service, API, dan aplikasi jaringan ringan sampai menengah;
- bukan tempat utama untuk cryptography atau observability.
