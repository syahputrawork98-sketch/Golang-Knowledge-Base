# BK-01: Web API with `net/http`

Buku ini membahas fondasi web API yang tersedia langsung di standard library Go, terutama server, client, handler, dan siklus request-response dengan package `net/http`.

## Struktur

### [CH-01_HTTP](./CH-01_HTTP/)
Dasar `net/http` untuk membangun endpoint sederhana, mengirim request, dan memahami aliran HTTP di Go.

## Boundary

- fokus pada kemampuan HTTP bawaan tanpa framework eksternal;
- membantu pembaca memahami kontrak dasar server dan client di Go;
- bukan tempat utama untuk routing kompleks, middleware framework, atau observability web yang lebih dalam.
