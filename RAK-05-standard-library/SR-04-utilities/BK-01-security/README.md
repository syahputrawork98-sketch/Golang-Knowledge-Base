# BK-01: Security Utilities

Buku ini membahas utilitas keamanan dasar yang tersedia langsung di standard library Go, terutama hashing dan enkripsi simetris untuk kebutuhan aplikasi sehari-hari.

## Struktur

### [CH-01_Crypto](./CH-01_Crypto/)
Dasar hashing, enkripsi simetris, dan pemakaian package `crypto/*` yang paling relevan untuk engineer aplikasi.

## Boundary

- fokus pada utilitas keamanan bawaan yang praktis dan sering dipakai;
- membantu pembaca memahami perbedaan tujuan hashing dan encryption;
- bukan tempat utama untuk protokol keamanan besar, TLS mendalam, atau cryptography tingkat riset.
