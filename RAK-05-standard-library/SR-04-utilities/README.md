# SR-04: Utilities

Sub-rak ini membahas utilitas penting di standard library Go yang sering dipakai untuk waktu, keamanan, dan kebutuhan aplikasi sehari-hari.

## Struktur

### [BK-01-security](./BK-01-security/)
Hashing, encryption, dan dasar cryptography dari package bawaan Go.

### [BK-02-time](./BK-02-time/)
Timer, ticker, dan manajemen waktu dalam program Go.

## Boundary

- fokus pada utilitas yang sering dipakai lintas jenis aplikasi;
- menolong pembaca memahami paket kecil tapi penting di standard library;
- bukan tempat utama untuk I/O atau web API.
