# BK-02: Time Utilities

Buku ini membahas utilitas waktu di standard library Go, terutama representasi waktu, durasi, serta primitive penjadwalan seperti timer dan ticker.

## Struktur

### [CH-01_Time](./CH-01_Time/)
Pemakaian package `time` untuk pengukuran durasi, scheduling sederhana, dan pengelolaan event berbasis waktu.

## Boundary

- fokus pada waktu dan scheduling dasar yang sering muncul di aplikasi Go;
- membantu pembaca memahami relasi `Time`, `Duration`, `Timer`, dan `Ticker`;
- bukan tempat utama untuk cron system, distributed scheduling, atau observability latency mendalam.
