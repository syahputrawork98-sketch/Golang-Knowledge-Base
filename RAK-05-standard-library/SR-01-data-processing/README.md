# SR-01: Data Processing

Sub-rak ini membahas cara standard library Go memproses teks, bytes, dan serialisasi data secara efisien.

## Struktur

### [BK-01-string-text](./BK-01-string-text/)
Manipulasi string, bytes, dan transformasi teks.

### [BK-02-serialization](./BK-02-serialization/)
JSON, binary encoding, dan bentuk serialisasi data yang umum dipakai.

## Boundary

- fokus pada pengolahan data menggunakan paket bawaan Go;
- relevan untuk aplikasi yang banyak berurusan dengan teks dan format data;
- bukan tempat utama untuk filesystem atau networking.
