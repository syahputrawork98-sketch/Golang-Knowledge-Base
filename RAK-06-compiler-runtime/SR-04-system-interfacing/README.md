# SR-04: System Interfacing

Sub-rak ini membahas batas interaksi antara Go dengan sistem luar, termasuk bahasa lain dan assembly.

## Struktur

### [BK-01-assembly-cgo](./BK-01-assembly-cgo/)
Cgo, assembly, dan mekanisme perbatasan antara Go dengan lapisan sistem rendah.

## Boundary

- fokus pada interfacing dan titik pertemuan Go dengan dunia luar;
- relevan untuk pembaca yang mulai menyentuh integrasi rendah atau optimisasi spesifik;
- bukan tempat utama untuk pipeline compiler umum atau garbage collection.
