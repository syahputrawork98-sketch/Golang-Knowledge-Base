# SR-03: Tooling Excellence

Sub-rak ini membahas tooling yang memperkuat cara kerja Go di level engineering: dependency management, benchmarking, profiling, dan praktik pengukuran kualitas teknis.

## Struktur

### [BK-01-dependency-management](./BK-01-dependency-management/)
Go modules, dependency workflow, dan pengelolaan dependensi dari sudut pandang praktik engineering.

### [BK-02-performance-benchmarking](./BK-02-performance-benchmarking/)
Benchmarking, profiling strategies, dan fuzz testing untuk mengukur serta memperkuat perilaku program.

## Boundary

- fokus pada alat dan praktik yang membuat workflow Go tetap tajam dan terukur;
- menjembatani teori desain dengan eksekusi engineering sehari-hari;
- bukan tempat utama untuk tutorial dasar tooling CLI atau observability produksi penuh.
