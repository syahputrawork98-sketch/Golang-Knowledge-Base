Buku ini membahas bagaimana Go menjaga dependency workflow tetap sederhana, reproducible, dan mudah dirawat lewat sistem module modern.

## Struktur

- bukan tempat utama untuk sejarah panjang `GOPATH` atau runtime package loading internals.
